# Discord Bot Template for Railway

## Files
- bot.py — mã bot
- requirements.txt — thư viện Python
- Procfile — Worker định nghĩa lệnh chạy

## Deploy lên Railway
1. Upload toàn bộ project lên GitHub
2. Vào Railway
3. Thêm Environment Variable:
   DISCORD_TOKEN = <token bot của bạn>
4. Deploy
5. Xem Logs để kiểm tra bot đã online hay chưa
